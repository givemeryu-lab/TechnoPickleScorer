# 🏓 피클볼 트래커 사용 매뉴얼

피클볼 복식 경기의 점수를 정식 사이드아웃(Side-out) 방식으로 기록하는 스코어 트래커입니다.

---

## 1. 화면 구성 한눈에 보기

```
┌─────────────────────────────────────┐
│ ☰   🏓 피클볼  세트 1/3   🔊 🏆 🕘 🔄 │  ← 헤더
│     팀 A 0 : 0 팀 B · 11점 2점차     │
├─────────────────────────────────────┤
│  선수1 / 선수2       0                │
│  선수3 / 선수4       0                │  ← 점수판 / 서브 안내
│      OOO 서브 · 팀 A-2               │
│           [세트 진행 칩]              │
├─────────────────────────────────────┤
│                                       │
│         🎾 코트 다이어그램            │
│      ⇄A   ⇆팀위치   ⇄B              │  ← 코트 위 위치교체 버튼
│                                       │
├─────────────────────────────────────┤
│   [ 팀 A 득점 ]   [ 팀 B 득점 ]       │  ← 탭존
├─────────────────────────────────────┤
│              [ ↩ 되돌리기 ]           │
└─────────────────────────────────────┘
```

---

## 2. 경기 시작하기

1. 좌측 상단 **☰(햄버거) 아이콘**을 눌러 **경기 세팅** 화면을 엽니다.
   - 점수 입력 버튼이 비활성 상태(경기 시작 전)일 때 탭존을 눌러도 자동으로 세팅 화면이 열립니다.
2. **팀 A / 팀 B** 선수 이름을 입력합니다.
   - 입력창 옆 🔍 아이콘을 누르면 최근에 사용한 선수 이름(로스터)에서 선택하거나 삭제할 수 있습니다.
3. **첫 서브 선수**를 4명 중 한 명 선택합니다. (배치: A / B / A / B 순서)
4. **목표 점수**(기본 11점), **승리 조건**(2점 차 또는 점수만), **시리즈**(1선승 / 2선승 / 3선승)를 설정합니다.
5. **▶ 경기 새로 시작** 버튼을 누르면 첫 세트가 시작됩니다.

> 저장된 과거 경기 기록을 보려면 세팅 화면 맨 아래 **🗂️ 저장된 경기 기록 보기(DB)** 버튼을 누르세요.

---

## 3. 득점 방식 — 사이드아웃 스코어링

이 앱은 정식 피클볼 규칙(사이드아웃 스코어링)을 따릅니다.

- **서브권을 가진 팀만 득점**할 수 있습니다.
- 서브권이 없는 팀이 랠리를 이기면 **점수는 그대로**, **서브권만 상대팀으로 이동**합니다 (사이드 아웃).
- 각 세트 시작 시, 시작 팀은 서버 한 명만 서브 기회를 갖고(사이드아웃 예외), 점수는 **0-0-2**로 표시됩니다. 첫 사이드아웃이 일어난 뒤부터는 정상적으로 팀마다 두 명의 서버(1번·2번)가 번갈아 서브합니다.
- 점수 콜은 **[서브팀 점수]-[리시브팀 점수]-[서버 번호]** 순서로 코트 화면과 토스트 팝업에 표시됩니다.

### 득점/사이드아웃 입력 방법
- 화면 하단 **탭존**에서 방금 랠리를 이긴 팀을 누릅니다.
  - 서브권을 가진 팀을 누르면 → 득점 + 서브 위치 좌우 교체
  - 서브권이 없는 팀을 누르면 → 사이드아웃(서브권 이동), 점수 변화 없음
- 점수가 바뀔 때마다 화면 상단에 **점수 토스트**(예: `1-1-2`)가 2초간 표시되고, 음성 안내(🔊 켜짐 시)가 나옵니다.
- 새 토스트가 뜨면 이전 토스트는 즉시 사라지고 새 팝업으로 교체됩니다.

---

## 4. 코트 화면 & 선수 위치 교체

코트 다이어그램 중앙에 3개의 버튼이 있습니다.

| 버튼 | 기능 |
|---|---|
| ⇄ (파란색, 왼쪽) | **A팀 선수 위치 교체** — A팀 두 선수의 좌/우 위치를 바꿉니다 |
| ⇆ (가운데, 큰 버튼) | **팀 위치 교체** — 코트에서 A팀/B팀이 표시되는 좌/우 위치를 통째로 바꿉니다 |
| ⇄ (금색, 오른쪽) | **B팀 선수 위치 교체** — B팀 두 선수의 좌/우 위치를 바꿉니다 |

- 세 버튼 모두 세트 시작 전/진행 중 상관없이 언제든 사용할 수 있습니다.
- **팀 위치 교체**를 누르면 하단 **탭존**의 팀 A/B 버튼 좌우 순서도 코트와 똑같이 바뀝니다.
- 현재 서브 중인 선수는 공 아이콘과 함께 강조 표시되고, 그 위로 점수 콜(예: `3-2-1`)이 나타납니다.

---

## 5. 세트/시리즈 진행

- 한 세트가 끝나면 **"OOO 세트 승리!"** 화면이 뜨고, **다음 세트 첫 서브 선수**를 고를 수 있습니다.
  - 이긴 팀의 상대 팀 선수 중 한 명을 선택한 뒤 **▶ 다음 세트** 버튼을 누르면 다음 세트가 시작됩니다.
- 세트 진행 칩(세트 1 / 세트 2 / 세트 3 …)으로 현재까지의 세트 스코어를 한눈에 볼 수 있습니다.
- 설정한 시리즈 방식(1/2/3선승)에 따라 필요한 세트를 먼저 이기면 시리즈가 종료되고 **Match Detail(경기 결과)** 화면이 자동으로 뜹니다.

---

## 6. 경기 결과 화면 (Match Detail)

시리즈가 끝나면 우승팀 배너와 함께 3개 탭이 표시됩니다.

- **상세**: 세트별 스코어와 포인트 수
- **통계**: 세트 승리, 총 득점, 득점률, 최고 세트 점수 등
- **타임라인**: 세트별 득점 흐름(실제 득점만 표시, 사이드아웃은 제외)

하단 버튼:
- **🔄 새 시리즈 시작**: 선수 이름/설정(목표 점수, 승리 조건, 시리즈 방식)은 유지한 채 새 경기를 시작합니다.
- **📤 경기결과 공유**: 세트 스코어와 통계를 포함한 텍스트를 카카오톡 등으로 바로 공유하거나 클립보드에 복사합니다.
- **닫기**: 결과 화면을 닫고 현재 화면으로 돌아갑니다.

---

## 7. 경기 기록 히스토리 (DB)

헤더의 🕘(시계) 아이콘 또는 세팅 화면의 **🗂️ 저장된 경기 기록 보기** 버튼으로 진입합니다.

- 지금까지 완료된 모든 시리즈 결과가 목록으로 저장되어 있습니다 (Firebase에 자동 동기화).
- 목록에서 항목을 누르면 Match Detail과 동일한 **상세 / 통계 / 타임라인** 탭과 **📤 경기결과 공유** 버튼을 확인할 수 있습니다.
- 각 항목을 삭제할 수 있는 삭제 버튼도 제공됩니다.

## 8. 헤더 아이콘 모음

| 아이콘 | 기능 |
|---|---|
| ☰ | 경기 세팅 화면 열기 |
| 🔊 / 🔇 | 음성 점수 안내 켜기/끄기 |
| 🏆 (시리즈 종료 시에만 표시) | Match Detail(경기 결과) 다시 열기 |
| 🕘 | 저장된 경기 기록(히스토리) 열기 |
| ↺ | 현재 경기 초기화 (확인창 후 모든 기록이 초기화됨) |

## 9. 기타 기능

- **↩ 되돌리기**: 방금 입력한 득점/사이드아웃을 한 단계 취소합니다. 세트가 막 끝난 직후에도 되돌릴 수 있습니다.
- **화면 켜짐 유지 / 동기화 상태**: 화면 우측 하단에 함께 표시됩니다.
  - 🔆 화면 켜짐 유지: 경기 중 화면이 꺼지지 않도록 유지 중인지 표시
  - ✓ 동기화됨 / ↑ 저장 중 / ⚡ 오프라인: 서버(Firebase) 연결 및 저장 상태
- **카카오톡 인앱 브라우저 자동 회피**: 카카오톡 앱 내에서 링크를 열면 자동으로 기본 브라우저(Chrome/Safari 등)로 이동해 실행됩니다.
- **선수 로스터**: 한 번 입력한 선수 이름은 자동으로 저장되어 다음에 이름 입력창 옆 🔍 버튼으로 빠르게 재사용할 수 있습니다.

## 10. 빠른 참고 — 3자리 점수 콜 읽는 법

예: **5-3-2** → "서브팀 5점, 리시브팀 3점, 서버 2번이 서브 중"

- 세트 시작 직후에는 항상 `0-0-2`로 시작합니다.
- 서브팀이 득점하면 숫자만 올라가고 서버 번호는 유지됩니다.
- 서브팀이 랠리에서 지면(사이드아웃) 점수는 그대로, 서버가 상대팀으로 넘어가며 번호가 바뀝니다.
